## 🚀 Igniting the Sanctuary Locally

If you’re technically capable (looking at you, CalPal devs), here is how you run this masterpiece:

1.  **Clone the Vault**:
    ```bash
    git clone https://github.com/JoyTheSloth/fitnessbunny.git
    ```
2.  **Fuel the Engine**:
    ```bash
    npm install
    ```
3.  **Configure the Crystals**:
    Create a `.env` file with your tokens:
    ```env
    NEXT_PUBLIC_GROQ_API_KEY=your_key
    NEXT_PUBLIC_SUPABASE_URL=your_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
    GOOGLE_CLIENT_ID=your_id
    GOOGLE_CLIENT_SECRET=your_secret
    NEXTAUTH_SECRET=your_secret
    ```
4.  **Launch the Sanctuary**:
    ```bash
    npm run dev
    ```
